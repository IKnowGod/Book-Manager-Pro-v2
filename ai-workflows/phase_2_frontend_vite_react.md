# Phase 2: Frontend UI & Layout (Premium Design)

## Objective
Build a modern, premium user interface using React and Vanilla CSS, focusing on a high-end authoring experience.

## Tasks

### 2.1. Global Design System
- Implement `index.css` with a color palette (Glassmorphism, Dark mode ready).
- Define typography tokens using modern fonts (Inter/Outfit).

### 2.2. Shell & Sidebar
- Build a responsive `Layout` component.
- Implement a persistent `Sidebar` for Book navigation.

### 2.3. Note Editor & Views
- Create the Note list view for a selected Book.
- Implement a rich-text or clean markdown-style editor for Note content.
- Integrate the Tagging UI (Tag suggestions and manual tagging).
- Implement a list and grid layout toggle for notes so the user can control their authoring view.

## Deliverables
- "Wow" factor UI with smooth animations.
- Fully interactive dashboard and note editor.
